# AWS Cloud Practitioner Foundations

This repository documents my hands-on learning, mini projects, and case studies
while preparing for the AWS Certified Cloud Practitioner (CLF-C02) exam.

## Purpose

- Understand AWS core services and cloud concepts
- Learn from a business and cost perspective
- Practice AWS Free Tier safely
- Build strong fundamentals before moving to Solutions Architect

## What's Included

- CP-level mini projects performed using AWS Management Console
- Business-oriented case studies
- Notes on pricing, security, and AWS best practices

## What This Repository Is NOT

- Not a production-grade architecture
- Not Infrastructure as Code
- Not advanced DevOps automation

This repository focuses on **clarity over complexity**.
